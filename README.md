# Noxe - Modern Movie Platform

Noxe is a sleek, responsive movie web application built with React and Vite. It features a modern dark-themed UI and a custom authentication system to provide a premium cinematic experience.

---

## Project Overview
This project was built to demonstrate a modern frontend workflow, focusing on user experience, clean UI components, and client-side data persistence.

## Key Features
* Authentication System: Fully functional Login and Register pages that use LocalStorage for data persistence.
* Modern UI/UX: Designed with Tailwind CSS, featuring a unique cinematic tilted poster grid and smooth transitions.
* Fully Responsive: Optimized for all screen sizes (Mobile, Tablet, and Desktop).
* Fast Performance: Powered by Vite for near-instant hot module replacement (HMR).

## Tech Stack
* Frontend Library: React.js
* Build Tool: Vite
* Styling: Tailwind CSS
* Routing: React Router DOM
